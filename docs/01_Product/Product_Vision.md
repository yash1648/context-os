# Product Vision

## Vision Statement

**ContextOS** is an AI-powered Personal Operating System that unifies every dimension of a user's digital life into a single, intelligent context layer. It transforms fragmented information — books, movies, courses, projects, goals, habits, notes, and knowledge assets — into a rich, interconnected knowledge graph that the user can query, explore, and act upon.

We envision a world where your digital life has a single source of truth. ContextOS understands what you are reading, what you are building, what you are learning, and what matters to you — and surfaces the right information at the right time, proactively.

## Core Philosophy

> **Everything is a Container.**

Every entity in the user's digital life — a book, a movie, a software project, a learning goal, a habit — is modeled as a **Container**. Containers are the universal primitive. They hold metadata, track progress, record timelines, store snapshots, carry tags, and maintain AI context. This unifying abstraction allows the system to treat all knowledge uniformly while preserving domain-specific semantics through configurable types.

## Long-Term North Star

ContextOS aims to evolve through five major phases:

1. **V1 — MVP**: Personal container management with manual context tracking
2. **V2 — AI Integration**: Automated context extraction, semantic search, and recommendations
3. **V3 — Browser Extension**: Capture web content directly into containers
4. **V4 — VSCode Integration**: In-editor code context awareness for developers
5. **V5 — Personal OS**: Full operating system layer for digital life management

## Target Audience

- **Knowledge Workers** — Researchers, writers, analysts managing large information footprints
- **Software Engineers** — Developers tracking projects, learning, and technical reading
- **Lifelong Learners** — Individuals managing courses, books, and skill development
- **Productivity Enthusiasts** — Users who track goals, habits, and personal growth metrics
- **Power Users** — Anyone who wants AI-powered context retrieval across their digital life

## Key Differentiators

| Differentiator | Description |
|---|---|
| Universal Container Model | Every entity is a Container — consistent API, storage, and query patterns |
| AI-Native Architecture | Built from the ground up with embeddings, vector search, and RAG |
| Semantic Context Layer | Not just storage — the system *understands* relationships between items |
| Progressive Enhancement | Start as a note-taking tool, evolve into a full personal OS |
| Open & Extensible | Plugin architecture enables custom container types and integrations |
